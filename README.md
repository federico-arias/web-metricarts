Para desarrollar:

1. Clona este repositorio 
2. ```npm install```
3. ```npm start```
4. Anda a ```localhost:8080``` en tu navegador.

Para crear un tema de wordpress:

1. Clona este repositorio 
2. ```npm install```
3. ```npm run wordpress```
4. La carpeta ```target``` contiene los archivos para el tema de Wordpress.
